# Drowsiness detection example using opencv haar-cascade files

A simple drowsiness detection algorithm in python that uses the vertical position of the eyes in a period of time to detect if the user is dozing off.

# Um exemplo de detecção de sonolência usando os arquivos haar-cascade do opencv

Um simples algoritmo de detecção de sonolência em python que usa a posição vertical dos olhos em um período de tempo para detectar se o usuário está pescando ou não.
